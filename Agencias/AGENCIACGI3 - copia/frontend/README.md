# Frontend - CGI Autos

Contenido estático de la página web de CGI Autos.

## Estructura

- `index.html` - Página principal
- `admin.html` - Página de administración
- `admin_clientes_satisfechos.html` - Página de testimonios de clientes
- `styles/` - Archivos CSS
- `scripts/` - Archivos JavaScript
- `fotos-autos/` - Imágenes de vehículos
- `clientes-satisfechos/` - Testimonios con fotos
- `manifest.json` - Configuración PWA
- `robots.txt` - Instrucciones para motores de búsqueda
- `sitemap.xml` - Mapa del sitio

## Despliegue en Netlify

1. Conecta este repositorio a Netlify
2. Configura la carpeta base como `frontend/`
3. No necesita comando de build (archivos estáticos)
4. Netlify servirá automáticamente desde esta carpeta

## Archivos estáticos

Todos los archivos HTML, CSS, JS, imágenes y otros recursos se sirven directamente desde esta carpeta.
